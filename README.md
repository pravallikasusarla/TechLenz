# TechLenz

An automated, end-to-end event and launch coverage platform designed to showcase, track, and register for cutting-edge technical milestones. TechLenz brings you real-time ecosystem insights with same-day delivery metrics.

## Live Demo
Check out the live deployment here: **[tech-lenz.vercel.app](https://tech-lenz.vercel.app)**

---

## Features

* **Interactive Event Exploration:** Filter real-world technical events by timeline, city tracking, industry segment (AI/ML, Web3, SaaS), and custom event structures.
* **Fluid UI/UX:** Responsive layouts equipped with custom scroll animations, card hover effects, and crisp interface elements.
* **Direct Event Actions:** Seamless integration for users to register directly or explore event sources directly via specialized call-to-actions.
* **Integrated Support:** Native chatbot interface designed to assist users natively on the platform.

---

## Tech Stack

* **Frontend Framework:** React 19 (Vite)
* **Styling:** Custom Vanilla CSS Modules (Flexbox, CSS Grid, Custom Keyframes)
* **Deployment & Hosting:** Vercel Continuous Integration & Deployment
